# Teacher Name: 		Alaa Mostafa  
Course Date: 			16/02/2026 – 10/03/2026  
Course Name: 			Data Technician  
Combined Learning Summary: 	Weeks 1–8  

# Summary of What We Learned Across All Weeks

---

### Week 1 – Excel Foundations and Data Laws  
We began the programme by learning the essential laws that govern how organisations handle data, including GDPR, the Data Protection Act, the Freedom of Information Act, and the Computer Misuse Act. This built a strong foundation in ethical and legal responsibilities.  
We then developed core Excel skills such as sorting, filtering, using formulas, creating tables, and building pivot tables. We also explored data visualisation and learned how to present insights clearly. The week ended with a focus on communication skills, including how to present findings to senior leaders.

---

### Week 2 – Tableau, Power BI, and Data Visualisation  
We explored different versions of Tableau and learned how dashboards are used to communicate insights. Using real datasets, we created bar charts, maps, and interactive visuals.  
We then moved into Power BI, learning how to import data, transform it using Power Query, and design professional reports and dashboards. This week strengthened our understanding of business intelligence tools and how organisations use them to make data‑driven decisions.

---

### Week 3 – Databases and SQL Fundamentals  
We studied the structure of relational databases, including primary keys, foreign keys, and table relationships. We compared relational and non‑relational databases and explored when each is appropriate.  
We learned how SQL JOINs work and practised writing queries to combine and analyse data. We designed a database for a small retail business and completed a series of SQL practical tasks using the world database, strengthening our ability to extract insights using SQL.

---

### Week 4 – Careers Support & Professional Development  
This week focused on preparing for the workplace and understanding the expectations of data roles. We explored CV writing, interview preparation, professional communication, and how to present ourselves confidently in technical and non‑technical conversations.  
We also discussed career pathways within data, how to build a strong portfolio, and how to communicate our skills effectively to employers. This week helped us understand how to transition from training into employment.

---

### Week 5 – Cloud Computing and Azure Fundamentals  
We explored cloud computing concepts, including IaaS, PaaS, SaaS, and deployment models such as public, private, hybrid, and community cloud.  
We studied UK data protection laws and cybersecurity regulations, including the Computer Misuse Act and the Police and Justice Act.  
We completed Azure labs covering relational data, non‑relational data, and analytics services. We also completed the DP‑900 practice exam and produced a detailed Azure proposal for a fictional business, applying cloud concepts to a real‑world scenario.

---

### Week 6 – Python Programming and Data Analysis  
We strengthened our Python fundamentals through exercises such as FizzBuzz and then moved into data analysis using Pandas.  
We learned how to load, explore, manipulate, and visualise data using DataFrames. We practised indexing, slicing, grouping, pivot tables, and exporting data.  
We applied these skills to real datasets, including student performance and GDP data, and completed a group notebook activity to reinforce our learning.

---

### Week 7 – GitHub and Version Control  
We spent a full week learning how to use GitHub to manage and share our work.  
We covered:  
- Creating and organising repositories  
- Cloning, committing, and pushing changes  
- Understanding branches and pull requests  
- Writing professional README files  
- Uploading project files and building a portfolio  
- Using GitHub for collaboration and version control  

This week helped us build confidence in using GitHub as a central place to store and present our work, preparing us for real‑world workflows used in data and software teams.

---

### Week 8 – Graduation Week: Elevator Pitch & Final Presentation  
We ended the programme by preparing and delivering our **elevator pitches** and **final presentations**.  
We focused on:  
- Communicating our skills clearly and confidently  
- Presenting our projects and portfolios  
- Explaining our learning journey across all eight weeks  
- Practising professional speaking skills  
- Receiving feedback to prepare for real interviews  

Graduation week brought together everything we learned, helping us articulate our strengths and present ourselves as job‑ready data professionals.

---

## Overall Learning  
Across all eight weeks, we developed a strong, well‑rounded foundation in data, analytics, cloud, programming, and professional skills:

- Understanding data laws, ethics, and cybersecurity  
- Building Excel skills for analysis and reporting  
- Creating dashboards in Tableau and Power BI  
- Writing SQL queries and designing relational databases  
- Understanding cloud computing and Azure services  
- Using Python and Pandas for data manipulation and visualisation  
- Managing projects and portfolios using GitHub  
- Developing communication, presentation, and career‑readiness skills  

These combined skills reflect the core responsibilities of a modern Data Technician and prepare us for real‑world data roles, apprenticeships, and further technical development.
