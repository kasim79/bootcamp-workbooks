# Summary of What We Did and What We Learned

### Day 1 – Understanding Tableau and Creating Dashboards  
We began the week by researching the different versions of Tableau and comparing their features. This helped build an understanding of how data visualisation tools differ and why organisations choose specific versions. We also explored the limitations of Tableau Public, including restricted data connections and the requirement to publish dashboards publicly.  
Using the EMSI Job Change dataset, we created a dashboard that included a bar chart showing percentage change and a UK map highlighting key city locations. This introduced dashboard design, combining visuals, and presenting insights clearly.

---

### Day 2 – Analysing Spotify and Health Data  
We analysed the Spotify dataset to identify trends, patterns, and insights that could support future business decisions. This open‑ended task encouraged independent analysis, critical thinking, and storytelling with data.  
Next, we worked with a Health dataset to explore how data can support decision‑making in the NHS. We reflected on how data can highlight risks, improve patient outcomes, and guide resource planning. This reinforced the real‑world value of data in public services and healthcare.

---

### Day 3 – Power BI: Getting and Transforming Data  
We completed Lab 1, learning how to import data into Power BI Desktop. This introduced the first steps of building a BI report and understanding how Power BI connects to different data sources.  
In Lab 2, we transformed and loaded data using Power Query. This developed skills in cleaning, shaping, and preparing data before analysis. These tasks built confidence in managing data pipelines and ensuring accuracy in reporting.

---

### Day 4 – Power BI: Designing Reports and Dashboards  
We completed Lab 8, designing a Power BI report using visuals, formatting, and layout principles. This developed our ability to communicate insights clearly and professionally.  
Finally, we completed Lab 12, creating a Power BI dashboard. This brought together all skills from the week—data import, transformation, visualisation, and storytelling—into a single interactive dashboard suitable for business use.

---

## Overall Learning  
Throughout Week 2, we developed strong skills in business intelligence and data visualisation:

- Understanding the differences between Tableau versions and their use cases  
- Creating dashboards using real datasets  
- Identifying trends and insights in open‑ended analysis  
- Using data to support decision‑making in organisations like the NHS  
- Importing, transforming, and modelling data in Power BI  
- Designing clear, professional reports and dashboards  

These activities strengthened both technical skills and analytical thinking, preparing us for more advanced BI and data analysis work.
