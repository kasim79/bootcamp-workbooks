# Summary of What We Did and What We Learned

### Day 2 – Python Fundamentals and FizzBuzz  
We began the week by practising core Python programming skills through the classic **FizzBuzz** challenge. This task helped reinforce key concepts such as loops, conditional statements, and logical operators. By writing and testing the FizzBuzz program, we strengthened our understanding of Python syntax and built confidence in writing clean, functional code.  
This exercise also reflected a common interview question, helping us understand how Python fundamentals are assessed in real‑world technical roles.

---

### Day 3 – Working with Pandas and CSV Data  
We used the *student.csv* dataset to explore the Pandas library and develop essential data‑handling skills. Through a series of structured exercises, we learned how to:

- Load CSV files into a DataFrame  
- Inspect and explore data using `.head()`, `.info()`, and `.describe()`  
- Select columns and rows using indexing and slicing  
- Filter data based on conditions  
- Add, rename, and drop columns  
- Group and aggregate data  
- Create pivot tables  
- Sort values  
- Export DataFrames to new CSV files  

These exercises helped us understand how Python is used for real‑world data analysis and how Pandas simplifies working with structured datasets.

---

### Day 3 – Advanced Data Operations and Visualisation  
We extended our skills by performing more advanced operations such as creating calculated columns, categorising data using `pd.cut()`, and generating pivot tables for deeper insights.  
We also created simple visualisations using Pandas’ built‑in plotting functions, helping us understand how to quickly explore trends and patterns in data. This introduced the foundations of data visualisation in Python, preparing us for more advanced plotting libraries in the future.

---

### Day 4 – GDP Dataset Analysis  
We worked individually with the *GDP (nominal) per Capita* dataset to practise reading, exploring, and analysing real‑world economic data. We:

- Loaded the dataset into a DataFrame  
- Displayed the first and last rows  
- Selected specific columns  
- Explored the structure and meaning of the data  

This task helped us apply our Pandas skills to a new dataset and reinforced the importance of understanding data context before analysis.

---

### Day 4 – Group Python Activity and Open Exploration  
We worked through the *Day_4_Python_Activity.ipynb* notebook as a group, answering questions and experimenting with the data. This encouraged collaboration, problem‑solving, and peer learning.  
After completing the structured tasks, we explored additional datasets to practise our skills further. This open‑ended activity allowed us to experiment with filtering, grouping, visualising, and manipulating data, helping us build confidence and creativity in Python.
